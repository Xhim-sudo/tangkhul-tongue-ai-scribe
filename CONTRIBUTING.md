# Contributing to Tangkhul Translator

Thank you for your interest in contributing to the Tangkhul Translator! This project helps preserve and promote the Tangkhul language through community-powered translations.

## Getting Started

1. Fork the repository
2. Clone your fork: `git clone https://github.com/YOUR_USERNAME/tangkhul-tongue-ai-scribe.git`
3. Install dependencies: `npm install`
4. Start the dev server: `npm run dev`
5. Open `http://localhost:8080`

### Code Contributions
1. Create a feature branch: `git checkout -b feature/your-feature`
2. Make your changes
3. Run type checking: `npx tsc --noEmit`
4. Build successfully: `npm run build`
5. Commit with a clear message: `git commit -m "feat: add new feature"`
6. Push and open a Pull Request

### Reporting Issues
Use the GitHub Issues tab with the provided templates for bugs or feature requests.
